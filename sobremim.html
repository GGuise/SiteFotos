Esse site é um projeto pessoal criado para compartilhar minhas fotos esféricas de viagens.

Todas as fotos são capturadas por drone e processadas para proporcionar uma experiência de visualização perfeita diretamente no seu navegador.

Tecnologias Utilizadas

  • Visualizador de fotos esféricas para renderização de imagens em 360°
  • Tailwind CSS para estilo responsivo
  • JavaScript puro para interatividade
